# AI_Powered_Focus_App

A minimal React Native (Expo) skeleton for a focus app:

- Add tasks and assign minutes per task
- Start a timed focus flow
- Automatically moves to the next task when the timer finishes
- Adds automatic break sessions between tasks
- Includes an adjustable focus-mode policy placeholder:
  - Work session: notifications blocked + non-essential apps blocked
  - Break session: notifications allowed

## Run locally

```bash
npm install
npm run web
```
